# Portfolio Setup Instructions

## Where to Add Your Files

### 1. Profile Photo
**Location:** `/public/profile-photo.jpg`

- Place your profile photo in the `public` folder
- File name should be: `profile-photo.jpg`
- Recommended size: 800x800px or larger (square format preferred)
- Supported formats: JPG, PNG, or WebP
- The photo will be displayed in the Hero section automatically

### 2. Resume
**Location:** `/public/resume.pdf`

- Place your resume PDF in the `public` folder
- File name should be: `resume.pdf`
- To add a download button, we can update the Contact or About section
- The resume will be accessible at: `https://yourdomain.com/resume.pdf`

## How to Add Resume Download Button

If you want to add a resume download button, I can update the Contact or Hero section to include a "Download Resume" button. Just let me know which section you prefer!
