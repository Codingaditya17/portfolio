# Public Assets

## Profile Photo

To add your profile photo:

1. Place your photo file in this directory (`/public/`)
2. Name it exactly: `profile-photo.jpg`
3. Recommended size: 800x800px or larger (square format preferred)
4. Supported formats: JPG, PNG, or WebP

The photo will automatically appear in the About section. If the photo doesn't exist, it will show your initials (AP) as a fallback.

## Resume

To add your resume:

1. Place your resume PDF in this directory (`/public/`)
2. Name it exactly: `Aditya_Patel_Resume.pdf`
3. The "Download Resume" button in the Hero section will automatically work
